Shopping List

The goal of this project was to make a shopping list tool that can add items, delete items, and clear all items. I accomplished this by making specific blocks tailored to the purpose of the sprite. Each sprite has a different purpose. A challenge was getting the input to work to delete the numbered items. I learned how to make blocks with similar, yet different functions.
